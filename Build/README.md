# Atlyss Execute HP Indicator
A true warrior senses when to Execute… but sometimes even true warriors appreciate a yellow warning light.

## Overview
Atlyss Execute HP Indicator shows you when an enemy’s HP drops below 25%, making it easier to use Execute.

## Installation
- Place the DLL directly inside your **plugins** folder  
**or**  
- Extract its folder into **plugins**

---

## Features
- Visual indicator for enemy Execute range
- Health bar turns yellow below 25% HP
- Helps timing Execute more reliably
- Works automatically for all enemies
